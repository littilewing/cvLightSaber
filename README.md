# cvLightSaber
Color tracking Light Saber with openframeworks and opencv



# Specs

- openframeworks 0.84
- addons
  - ofxCV
  - ofxBox2d
  - ofxXmlSettings
  - ofxOpenCV
  - ofxGui


- Xcode 6.4
- MacBook Pro (Retina, 13-inch, Mid 2014,FacecamHD:1280x720)
- MacBook Air (13-inch,Mid 2011,iSight:640x480)
