# cvLightSaber
Color tracking Light Saber with openframeworks and opencv



# Specs

- openframeworks 0.84
- addons
  - ofxCV
  - ofxBox2d
  - ofxXmlSettings
  - ofxOpenCV
  - ofxGui


- Xcode 6.4
- MacBook Pro (Retina, 13-inch, Mid 2014)
